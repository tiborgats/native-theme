# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.4.x   | Yes       |
| < 0.4   | No        |

## Reporting a Vulnerability

**Please do not file a public GitHub issue for security vulnerabilities.**

Instead, use one of these channels:

- **GitHub Private Vulnerability Reporting:** Go to the [Security tab](https://github.com/tiborgats/native-theme/security) and click "Report a vulnerability"
- **Email:** Contact the maintainer at the address listed on their [GitHub profile](https://github.com/tiborgats)

## Response Timeline

We aim to acknowledge reports within 48 hours and provide a fix or mitigation plan within 7 days.

## Security Auditing

[`cargo audit`](https://rustsec.org/) runs in CI via [rustsec/audit-check](https://github.com/rustsec/audit-check) to detect known vulnerabilities in dependencies.
