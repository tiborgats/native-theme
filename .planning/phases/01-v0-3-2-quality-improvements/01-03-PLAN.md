  <action>
Find the python3 usage around line 127-128:

```bash
WORKSPACE_CRATES=$(cargo metadata --no-deps --format-version 1 2>/dev/null \
    | python3 -c "import sys,json; [print(p['name']) for p in json.load(sys.stdin)['packages']]")
```

Replace with jq + bash fallback:

```bash
# Get all workspace crate names
if command -v jq &>/dev/null; then
    WORKSPACE_CRATES=$(cargo metadata --no-deps --format-version 1 2>/dev/null \
        | jq -r '.packages[].name')
else
    # Pure-bash fallback: cargo metadata outputs compact JSON (no spaces around colons)
    WORKSPACE_CRATES=$(cargo metadata --no-deps --format-version 1 2>/dev/null \
        | grep -o '"name":"[^"]*"' | sed 's/"name":"//;s/"//')
fi
```

Notes:
- `cargo metadata --format-version 1` outputs compact JSON, so the grep pattern `"name":"[^"]*"` is safe (no spaces around colons).
- The 3 workspace crate names (native-theme, native-theme-gpui, native-theme-iced) are simple ASCII names.
- Do NOT add jq to any "required tools" check at the top of the script since we have the bash fallback.
