Add `#[must_use = "..."]` with descriptive messages to the following. Use the exact wording below (per research recommendations):

**In `native-theme/src/lib.rs`:**
- `from_system()`: `#[must_use = "this returns the detected theme; it does not apply it"]`
- `from_system_async()`: `#[must_use = "this returns the detected theme; it does not apply it"]` (there may be multiple cfg-gated versions; annotate all)
- `load_icon()`: `#[must_use = "this returns the loaded icon data; it does not display it"]`

**In `native-theme/src/model/icons.rs`:**
- `system_icon_set()`: `#[must_use = "this returns the current icon set for the platform"]`
- `system_icon_theme()`: `#[must_use = "this returns the current icon theme name"]`

**In `native-theme/src/model/bundled.rs`:**
- `bundled_icon_svg()`: `#[must_use = "this returns SVG bytes; it does not render the icon"]`
- `bundled_icon_by_name()`: `#[must_use = "this returns SVG bytes; it does not render the icon"]`

**In `native-theme/src/model/mod.rs`:**
- On `pub struct NativeTheme`: `#[must_use = "constructing a theme without using it is likely a bug"]` (place before the struct definition, after any existing derive/attribute macros)
- On `NativeTheme::preset()`: `#[must_use = "this returns a theme preset; it does not apply it"]`
- On `NativeTheme::from_toml()`: `#[must_use = "this parses a TOML string into a theme; it does not apply it"]`
- On `NativeTheme::from_file()`: `#[must_use = "this loads a theme from a file; it does not apply it"]`
- On `NativeTheme::to_toml()`: `#[must_use = "this serializes the theme to TOML; it does not write to a file"]`
- On `NativeTheme::list_presets()`: `#[must_use = "this returns the list of preset names"]`
- On `NativeTheme::pick_variant()`: `#[must_use = "this returns the selected variant; it does not apply it"]`
- On `pub enum IconData` (in the icons module or wherever it's defined): `#[must_use = "loading icon data without using it is likely a bug"]`

**After adding annotations:** Run `cargo check --workspace` and check for any new warnings in test code. If any test creates a NativeTheme or IconData without using it, fix with `let _theme = ...` or add an assertion. Do NOT suppress warnings with `#[allow(unused)]` in production code.
