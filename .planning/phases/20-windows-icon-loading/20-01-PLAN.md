Purpose: Windows users get native platform icons matching their OS's visual language, with guaranteed fallback to bundled cross-platform icons.

fn unpremultiply_alpha(buffer: &mut [u8]) {
    for pixel in buffer.chunks_exact_mut(4) {
        let a = pixel[3] as u16;
        if a > 0 && a < 255 {
            pixel[0] = ((pixel[0] as u16 * 255) / a).min(255) as u8;
            pixel[1] = ((pixel[1] as u16 * 255) / a).min(255) as u8;
            pixel[2] = ((pixel[2] as u16 * 255) / a).min(255) as u8;
        }
    }
}

Functions to implement (in this order):

1. `fn glyph_codepoint(name: &str) -> Option<u32>` -- Match table mapping 22 glyph names to Unicode PUA codepoints. Use the exact codepoints from the research: ChromeClose=0xE8BB, ChromeMinimize=0xE921, ChromeMaximize=0xE922, ChromeRestore=0xE923, Save=0xE74E, Copy=0xE8C8, Paste=0xE77F, Cut=0xE8C6, Undo=0xE7A7, Redo=0xE7A6, Edit=0xE70F, Add=0xE710, Remove=0xE738, Refresh=0xE72C, Back=0xE72B, Forward=0xE72A, Up=0xE74A, Down=0xE74B, Home=0xE80F, GlobalNavigationButton=0xE700, CheckMark=0xE73E, Ringer=0xEA8F.

2. `fn siid_from_name(name: &str) -> Option<SHSTOCKICONID>` -- Match table mapping 16 SIID_ string names to their SHSTOCKICONID enum constants. SIID_WARNING, SIID_ERROR, SIID_INFO, SIID_SHIELD, SIID_DELETE, SIID_FIND, SIID_SETTINGS (use SIID_HELP for this -- research notes ambiguity, check if SIID_SETTINGS exists in the windows crate, if not use a reasonable substitute), SIID_PRINTER, SIID_DOCNOASSOC, SIID_FOLDER, SIID_FOLDEROPEN, SIID_RECYCLER, SIID_RECYCLERFULL, SIID_USERS, SIID_HELP, SIID_LOCK.

3. `fn bgra_to_rgba(pixels: &mut [u8])` -- Swap bytes 0 and 2 in each 4-byte chunk (B and R channels).

4. `fn unpremultiply_alpha(buffer: &mut [u8])` -- Same logic as sficons.rs: for each pixel where a > 0 && a < 255, scale channels by 255/a. Fully opaque and fully transparent pixels unchanged.

5. `unsafe fn hicon_to_rgba(hicon: HICON) -> Option<IconData>` -- Extract RGBA pixels from HICON:
   - Call `GetIconInfo(hicon, &mut icon_info)` to get hbmColor bitmap
   - Get dimensions via `GetObjectW(icon_info.hbmColor, size_of::<BITMAP>(), ...)`
   - Create compatible DC, set up BITMAPINFOHEADER with negative biHeight (top-down), 32-bit, BI_RGB
   - Call `GetDIBits` to extract pixel data
   - Clean up DC and bitmaps (DeleteDC, DeleteObject for hbmColor and hbmMask)
   - Apply bgra_to_rgba conversion
   - Apply unpremultiply_alpha
   - Return IconData::Rgba { width, height, data }

6. `fn load_stock_icon(name: &str) -> Option<IconData>` -- Map SIID name -> SHSTOCKICONID, call SHGetStockIconInfo with SHGSI_ICON | SHGSI_LARGEICON, extract via hicon_to_rgba, clean up with DestroyIcon.

7. `fn load_idi_icon() -> Option<IconData>` -- Special case for DialogQuestion: call LoadIconW(None, IDI_QUESTION) to get HICON, extract via hicon_to_rgba. Note: system icons from LoadIconW are shared resources and should NOT be destroyed with DestroyIcon.

8. `unsafe fn try_create_font(hdc: HDC, face_name: &str, size: i32) -> Option<HFONT>` -- Create font with CreateFontW, verify via GetTextFaceW that the actual loaded font matches the requested name (GDI silently substitutes). Return None if substituted.

9. `fn load_glyph_icon(codepoint: u32, size: i32) -> Option<IconData>` -- Font glyph pipeline:
   - Create compatible DC
   - Try Segoe Fluent Icons first, then Segoe MDL2 Assets via try_create_font
   - If neither font available, return None (caller will fall back to bundled)
   - Set up identity MAT2 matrix
   - Call GetGlyphOutlineW twice: first with 0 buffer to get size, then with buffer to get data
   - Calculate pitch as (glyph_width + 3) & !3 for DWORD alignment
   - Convert GGO_GRAY8_BITMAP (65-level grayscale 0-64) to RGBA: R=255, G=255, B=255, A=(gray * 255 / 64).min(255)
   - Clean up font and DC
   - Return IconData::Rgba { width: gmBlackBoxX, height: gmBlackBoxY, data }

10. `pub fn load_windows_icon(role: IconRole) -> Option<IconData>` -- Main entry point:
    - Get name from icon_name(IconSet::SegoeIcons, role)
    - If name starts with "SIID_": dispatch to load_stock_icon(name)
    - If name == "IDI_QUESTION": dispatch to load_idi_icon()
    - Otherwise: look up glyph_codepoint(name), dispatch to load_glyph_icon(codepoint, DEFAULT_ICON_SIZE)
    - If any pipeline returns None or name is None: fall back to bundled_icon_svg(IconSet::Material, role).map(|bytes| IconData::Svg(bytes.to_vec()))
