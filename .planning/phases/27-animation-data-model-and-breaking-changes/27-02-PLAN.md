Rename StatusLoading to StatusBusy across the entire workspace -- enum variant, ALL array, icon_name mappings, bundled icon lookups, connector references, tests, and examples.

Purpose: This is the breaking change in v0.4.0. StatusLoading was semantically wrong (a static icon cannot represent "loading"). The rename must be atomic -- every reference updated in one plan so the workspace compiles at every commit.
