Color Mapping Strategy (from 07-RESEARCH.md):
- core: accent, background, foreground, surface, border, muted, shadow
- primary: bg = accent, fg = white/on-accent
- secondary: bg = muted surface, fg = contrast text
- status: danger = red, warning = yellow/orange, success = green, info = blue/cyan
- interactive: selection = accent, link = blue, focus_ring = accent
- panel: sidebar = offset bg, tooltip = inverted, popover = surface
- component: button = elevated bg, input = surface/white, disabled = muted, separator = border

Catppuccin light/dark pairing (from 07-RESEARCH.md):
