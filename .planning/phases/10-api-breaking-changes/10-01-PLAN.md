<objective>
Flatten ThemeColors from 6 nested sub-structs to 36 direct Option<Rgba> fields, migrate all 17 preset TOML files to flat [light.colors]/[dark.colors] tables, and update all platform readers and tests.

Purpose: This is the foundation change for v0.2's API -- every subsequent plan depends on flat ThemeColors. The nested sub-struct access path (e.g., `colors.core.accent`) becomes the flat path (`colors.accent`), making the API simpler and the TOML format flatter.

Output: Compilable crate with flat ThemeColors, flat TOML presets, updated platform readers, all 140+ tests passing.
</objective>
