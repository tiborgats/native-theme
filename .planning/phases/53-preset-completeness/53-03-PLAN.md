- caption: size=12.0, weight=400 (iOS .caption1 is 12pt Regular)
- section_heading: size=15.0, weight=700 (iOS .headline is 15pt Bold on iOS, vs 13pt on macOS)

**Material preset (Material Design 3 type ramp):**
Read the material.toml to find its base font size (likely 14.0). Replace text_scale with MD3 values:
- caption: size=11.0, weight=400 (MD3 Label Small = 11sp, Regular)

- display: size=36.0, weight=400 (MD3 Display Small = 36sp, Regular)

**Catppuccin presets (all 4 flavors use GNOME/Adwaita text_scale):**
These are Linux-focused community themes. Use GNOME Adwaita values from platform-facts.md section 2.19:
- caption: size=9.0, weight=400 (GNOME .caption)
- section_heading: size=11.0, weight=700 (GNOME .heading -- same as defaults.font.size, differentiated by weight)

  <action>
Add ALL 51 interactive state color fields to both light and dark variants of all 6 presets.

**iOS preset -- derive from macOS patterns (Apple design language):**
Use the macOS column from platform-facts.md as a guide, adapting to iOS colors already in the preset. iOS uses the same design system as macOS (AppKit/UIKit share color semantics).

**Material preset -- derive from Material Design 3 color system:**
Material Design uses elevation/tonal overlays for state layers:
- Hover: 8% tonal overlay on container
- Active/Pressed: 12% tonal overlay
- Disabled container: 12% opacity of on-surface
- Disabled text: 38% opacity of on-surface
Use the material preset's existing palette to derive these overlays.

**Catppuccin presets -- derive from Catppuccin named palette colors:**
All 4 Catppuccin flavors share the same color role structure but with different values. The official palette provides named colors that map naturally to interactive states.

For each Catppuccin flavor, derive state colors from its palette:

Catppuccin palette mapping (consistent across all 4 flavors):
- hover_background: use Surface1 (one step above Base) or Overlay0 with alpha, e.g., `#surface180` (50% opacity of surface1 over base)
- active_background: use Surface2 or stronger overlay
- disabled_background: use Surface0 or Mantle with reduced alpha
- disabled_text_color: already set in defaults as disabled_text_color -- copy that value
- hover_text_color / active_text_color: same as font.color (most platforms don't change text on hover)
- Focus/hover border: use accent color (Blue/Sapphire)
- Unchecked checkbox: use Surface1/Surface2
- Scrollbar thumb hover: slightly brighter Overlay variant
- Arrow colors: use Subtext0 or Overlay2
