  description.svg, folder.svg, folder_open.svg,
  progress_activity.svg, check.svg,
  person.svg, notifications.svg, lock.svg

Note: Some IconRole variants share the same SVG file (e.g., DialogError and StatusError both use error.svg, TrashEmpty/TrashFull/ActionDelete all use delete.svg). Only 32 unique files are needed. The include_bytes! calls in Plan 02 will reference the same file for shared roles.
