Migration patterns:
- `bundled_icon_by_name(name, set)` (returns `Option<&'static [u8]>`) 
    -> `IconLoader::new(name).set(set).load()` (returns `Option<IconData>`) 
    -> match `IconData::Svg(cow)` -> `cow` is `Cow<'static, [u8]>`.
  When callers need `&[u8]`, do `&cow` or `cow.as_ref()`. When callers need `Vec<u8>`, do `cow.into_owned()`. Current `IconLoader.load()` for bundled sets internally wraps via `Cow::Borrowed(bytes)` so `cow.as_ref()` returns the same `&'static [u8]` without allocation.

- `load_freedesktop_icon_by_name(name, theme, size, fg)` (returns `Option<IconData>`) 
    -> `IconLoader::new(name).set(IconSet::Freedesktop).theme(theme).size(size).color_opt(fg).load()` (returns `Option<IconData>`).
