1. **Cargo.toml** -- Add three new feature flags to the existing `windows` dependency features list:
   - `"Win32_UI_HiDpi"` -- for GetSystemMetricsForDpi, GetDpiForSystem
   - `"Win32_Graphics_Gdi"` -- for LOGFONTW in font extraction
   - `"Foundation_Metadata"` -- for ApiInformation capability checks (PLAT-05)
   Keep all existing features (`UI_ViewManagement`, `Win32_UI_WindowsAndMessaging`).

2. **windows.rs** -- Enhance with the following new functions and changes:

   a. **`read_accent_shades(settings: &UISettings) -> [Option<Rgba>; 6]`** (new) -- Read AccentDark1, AccentDark2, AccentDark3, AccentLight1, AccentLight2, AccentLight3 from UISettings. Each call wrapped in `.ok().map(win_color_to_rgba)` for graceful fallback per PLAT-05. Order: [dark1, dark2, dark3, light1, light2, light3].

   b. **`read_system_font(dpi: u32) -> ThemeFonts`** (new) -- Read system font via `SystemParametersInfoW(SPI_GETNONCLIENTMETRICS, ...)`. Extract `lfMessageFont` from `NONCLIENTMETRICSW`. Parse `lfFaceName` as null-terminated UTF-16 `[u16; 32]` into a String via `String::from_utf16_lossy`. Convert `lfHeight` to points: `points = (lfHeight.unsigned_abs() * 72) / dpi`. Return ThemeFonts with family and size. mono_family and mono_size are None (Windows has no system monospace font setting). Return `ThemeFonts::default()` if SystemParametersInfoW fails. Use imports: `windows::Win32::UI::WindowsAndMessaging::{SystemParametersInfoW, NONCLIENTMETRICSW, SPI_GETNONCLIENTMETRICS, SYSTEM_PARAMETERS_INFO_UPDATE_FLAGS}`.

   c. **`winui3_spacing() -> ThemeSpacing`** (new) -- Return a ThemeSpacing with the WinUI3 Fluent Design defaults: xxs=2.0, xs=4.0, s=8.0, m=12.0, l=16.0, xl=24.0, xxl=32.0. Pure function, no OS API calls.

   d. **Replace `read_geometry()` with `read_geometry_dpi_aware() -> ThemeGeometry`** -- Use `GetDpiForSystem()` to get the system DPI, then `GetSystemMetricsForDpi(SM_CXBORDER, dpi)` and `GetSystemMetricsForDpi(SM_CXVSCROLL, dpi)` instead of plain `GetSystemMetrics`. Also populate `radius: Some(4.0)`, `radius_lg: Some(8.0)`, `shadow: Some(true)` (standard Windows 11 values). Returns the DPI value as well for font conversion: change signature to return `(ThemeGeometry, u32)`.

   e. **Expand `build_theme()`** signature to accept new data:
   ```rust
   fn build_theme(
       accent: Rgba,
       fg: Rgba,
       bg: Rgba,
       accent_shades: [Option<Rgba>; 6],  // [dark1, dark2, dark3, light1, light2, light3]
       fonts: ThemeFonts,
       spacing: ThemeSpacing,
       geometry: ThemeGeometry,
   ) -> NativeTheme
   ```
   In the body:
   - Keep existing: accent, foreground, background, selection, focus_ring mapped as before.
   - **primary_background**: In light mode use `accent_shades[0]` (AccentDark1) if available, else accent. In dark mode use `accent_shades[3]` (AccentLight1) if available, else accent.
   - **primary_foreground**: Derive from fg -- use the foreground color directly (PLAT-08). In light mode this is black-on-light (fg), in dark mode it's white-on-dark (fg). Since fg is already the system foreground, use `Some(fg)`.
   - **surface**: Same as bg (the window background).
   - **secondary_background**: Derive from bg with slight contrast -- but since Windows doesn't expose this, set to `Some(bg)`.
   - **secondary_foreground**: Same as fg.
   - **border**: Use a mid-gray default: in dark mode `Rgba::rgb(60, 60, 60)`, in light mode `Rgba::rgb(200, 200, 200)`.
   - Add fonts and spacing to the ThemeVariant.

   f. **Update `from_windows()`** to call the new functions:
   ```rust
   pub fn from_windows() -> crate::Result<NativeTheme> {
       let settings = UISettings::new().map_err(|e| ...)?;
       let accent = settings.GetColorValue(UIColorType::Accent).map(win_color_to_rgba).map_err(|e| ...)?;
       let fg = settings.GetColorValue(UIColorType::Foreground).map(win_color_to_rgba).map_err(|e| ...)?;
       let bg = settings.GetColorValue(UIColorType::Background).map(win_color_to_rgba).map_err(|e| ...)?;
       let accent_shades = read_accent_shades(&settings);
       let (geometry, dpi) = read_geometry_dpi_aware();
       let fonts = read_system_font(dpi);
       let spacing = winui3_spacing();
       Ok(build_theme(accent, fg, bg, accent_shades, fonts, spacing, geometry))
   }
   ```
