Create native-theme/src/sficons.rs following the exact pattern of freedesktop.rs:
- Module doc comment explaining purpose and fallback chain
- `use crate::{bundled_icon_svg, icon_name, IconData, IconRole, IconSet};`
- `use objc2::rc::Retained;` and imports from `objc2_app_kit`, `objc2_core_graphics`, `objc2_foundation`
- `const DEFAULT_ICON_SIZE: u32 = 24;` (matches freedesktop default)
- `fn load_symbol(name: &str, point_size: f64) -> Option<Retained<NSImage>>` -- creates NSImage from system symbol name, applies NSImageSymbolConfiguration with given point size, regular weight, medium scale
- `fn extract_cgimage(image: &NSImage) -> Option<Retained<CGImage>>` -- calls CGImageForProposedRect_context_hints with null rect/context/hints to get natural size CGImage
- `fn rasterize(cg_image: &CGImage, width: u32, height: u32) -> Option<Vec<u8>>` -- creates CGBitmapContext with PremultipliedLast alpha info, draws CGImage, returns raw buffer
- `fn unpremultiply_alpha(buffer: &mut [u8])` -- converts premultiplied RGBA to straight alpha. For each pixel where `a > 0 && a < 255`: `channel = min(255, channel * 255 / a)`. Skip fully opaque (a==255) and fully transparent (a==0) pixels.
- `pub fn load_sf_icon(role: IconRole) -> Option<IconData>` -- looks up SF Symbols name via `icon_name(IconSet::SfSymbols, role)`, loads symbol, extracts CGImage, reads actual pixel dimensions from CGImage::width()/height() (NOT NSImage size -- important for Retina), rasterizes, unpremultiplies, returns IconData::Rgba. Falls back to `bundled_icon_svg(IconSet::Material, role).map(|bytes| IconData::Svg(bytes.to_vec()))`.

Add inline tests module (`#[cfg(test)]`):
- `unpremultiply_correctness` -- pure unit test (runs on any platform): test with known premultiplied values [128, 0, 0, 128] -> [255, 0, 0, 128], fully opaque unchanged, fully transparent unchanged
- `#[cfg(target_os = "macos")]` gated tests:
  - `load_icon_returns_some` -- load_sf_icon(IconRole::ActionCopy) returns Some
  - `fallback_for_unmapped_role` -- load_sf_icon(IconRole::WindowRestore) returns Some (bundled fallback)
  - `rgba_dimensions_correct` -- if load_sf_icon returns Rgba, verify width * height * 4 == data.len()

IMPORTANT: Use `CGImageAlphaInfo::PremultipliedLast.0` for the bitmap info parameter (the `.0` extracts the raw u32 value). Use `CGContext::draw_image` as a method call. Use `CGColorSpace::new_device_rgb()` to create the color space.
