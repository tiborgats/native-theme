<!-- From Plan 01 (WidgetMetrics TOML format) -->

TOML table paths for widget metrics:
```toml
[light.widget_metrics.button]
min_width = 80.0
min_height = 34.0
padding_horizontal = 12.0
padding_vertical = 8.0
icon_spacing = 4.0

[light.widget_metrics.checkbox]
indicator_size = 20.0
spacing = 8.0

# ... etc for each of the 12 widget types
```

Only include sub-struct tables that have at least one non-None value.
Empty sub-structs (all fields None) should be OMITTED -- serde's
skip_serializing_if = "XxxMetrics::is_empty" handles this automatically,
but in hand-written TOML we simply don't write the table.
</interfaces>
</context>
