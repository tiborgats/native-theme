    - parse_sprite_sheet(b"<svg viewBox=\"0 0 10 20\">...</svg>") returns Some(vec![frame0_bytes, frame1_bytes]) where frame0 has viewBox="0 0 10 10" and frame1 has viewBox="0 10 10 10"
    - parse_sprite_sheet(b"<svg viewBox=\"0 0 22 330\">...</svg>") returns Some(15 frames) (Breeze-like 22x330 = 15 frames)
    - parse_sprite_sheet(b"<svg viewBox=\"0 0 22 22\">...</svg>") returns None (single-frame, height == width)
    - parse_sprite_sheet(b"<svg viewBox=\"0 0 22 33\">...</svg>") returns None (height not exact multiple of width)
    - parse_sprite_sheet(b"not svg") returns None (no viewBox found)
    - parse_sprite_sheet handles comma-separated viewBox values (viewBox="0,0,10,20")
    - load_freedesktop_spinner() returns Some(AnimatedIcon::Frames) when theme has a multi-frame process-working sprite sheet
    - load_freedesktop_spinner() returns Some(AnimatedIcon::Transform::Spin) when only a single-frame process-working icon exists
    - load_freedesktop_spinner() returns None when no process-working icon exists in the theme
  </behavior>
  <action>
Add two new functions to native-theme/src/freedesktop.rs:

1. `fn parse_sprite_sheet(svg_bytes: &[u8]) -> Option<Vec<Vec<u8>>>` (private pure function):
   - Convert bytes to UTF-8 string, return None on failure
   - Find `viewBox="..."` attribute (handle both double and single quotes)
   - Split the viewBox value on whitespace or commas, parse 4 f64 values (min_x, min_y, width, height)
   - If height <= width, return None (single-frame, not a sprite sheet)
   - Compute frame_count = (height / width).round() as usize; if < 2, return None
   - Verify exact division: (height - width * frame_count as f64).abs() < 0.01, else return None
   - Build the full original viewBox attribute string (e.g. `viewBox="0 0 22 330"`)
   - For each frame i in 0..frame_count, create a new SVG string by replacing the original viewBox attribute with `viewBox="0 {y_offset} {width} {width}"` where y_offset = width * i
   - Return Some(vec of frame byte vectors)

2. `pub(crate) fn load_freedesktop_spinner() -> Option<AnimatedIcon>` (crate-visible):
   - Call detect_theme() to get the active theme name
   - First pass: lookup "process-working" (plain name, finds sprite sheets in animations/ dirs) at size 22, force_svg
   - If found: read file bytes, try parse_sprite_sheet(). If it returns Some(frames), wrap in AnimatedIcon::Frames { frames: frames.into_iter().map(IconData::Svg).collect(), frame_duration_ms: 80, repeat: Repeat::Infinite } and return
   - If parse_sprite_sheet returns None (single-frame found under plain name), return AnimatedIcon::Transform { icon: IconData::Svg(bytes), animation: TransformAnimation::Spin { duration_ms: 1000 } }
   - Second pass: lookup "process-working-symbolic" at size 22, force_svg
   - If found: read file, return AnimatedIcon::Transform with Spin { duration_ms: 1000 }
   - If neither found: return None

Add imports at top of freedesktop.rs: `use crate::model::animated::{AnimatedIcon, Repeat, TransformAnimation};`

Add unit tests in the existing `#[cfg(test)]` module:
- test_parse_sprite_sheet_two_frames: 10x20 viewBox with 2 rectangles, verify 2 frames with correct viewBox values
- test_parse_sprite_sheet_fifteen_frames: 22x330 viewBox (Breeze-like), verify 15 frames
- test_parse_sprite_sheet_single_frame_returns_none: 22x22 viewBox returns None
- test_parse_sprite_sheet_non_multiple_returns_none: 22x33 viewBox returns None
- test_parse_sprite_sheet_invalid_svg_returns_none: non-SVG bytes return None
- test_parse_sprite_sheet_comma_separated_viewbox: viewBox="0,0,10,20" works correctly
- test_parse_sprite_sheet_preserves_svg_content: verify frame SVG content is identical except viewBox
- test_load_freedesktop_spinner_no_panic: call load_freedesktop_spinner(), verify no panic (result is theme-dependent)

Do NOT use a full XML parser -- string-level viewBox find-and-replace is sufficient and avoids new dependencies.
Do NOT look up at size 24 -- animated icons live at size 22 in freedesktop themes.
Do NOT cache the result in a static -- keep consistent with other loaders.
