Implement src/kde/fonts.rs:

1. `fn parse_qt_font(font_str: &str) -> Option<(String, f32)>`:
   - Split on ',' to get fields
   - Return None if fewer than 2 fields
   - Field [0] trimmed = family name; return None if empty
   - Field [1] trimmed parsed as f32 = point size; return None if parse fails or size <= 0.0
   - Return Some((family, size))
   - Do NOT validate or use fields beyond [0] and [1] -- they are Qt rendering hints we don't need

2. `pub(crate) fn parse_fonts(ini: &configparser::ini::Ini) -> crate::ThemeFonts`:
   - Get "General" / "font" key; if Some, parse with parse_qt_font -> set family and size
   - Get "General" / "fixed" key; if Some, parse with parse_qt_font -> set mono_family and mono_size
   - Return ThemeFonts (all None fields stay None if keys missing or unparseable)
