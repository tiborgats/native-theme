Update src/kde/mod.rs to replace the from_kde() stub with the full implementation:

1. Add an internal function `fn from_kde_content(content: &str) -> crate::Result<NativeTheme>` that:
   - Creates parser via create_kde_parser()
   - Parses content string via ini.read(content).map_err(|e| Error::Format(...))?
   - Calls colors::parse_colors(&ini) to get ThemeColors
   - Calls fonts::parse_fonts(&ini) to get ThemeFonts
   - Calls is_dark_theme(&ini) to determine variant
   - Extracts theme name: ini.get("General", "ColorScheme").unwrap_or_else(|| "KDE".to_string())
   - Builds ThemeVariant { colors, fonts, geometry: Default, spacing: Default }
   - Builds NativeTheme { name, light: if !dark Some(variant) else None, dark: if dark Some(variant) else None }
   - Returns Ok(theme)

2. Implement `pub fn from_kde() -> crate::Result<NativeTheme>`:
   - let path = kdeglobals_path();
   - let content = std::fs::read_to_string(&path).map_err(|e| Error::Unavailable(format!("cannot read {}: {e}", path.display())))?;
   - from_kde_content(&content)
