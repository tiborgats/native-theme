Create `native-theme/src/model/icons.rs` with:

1. **IconRole enum** -- Flat enum with prefixed naming per user's locked decision. Exactly 42 variants:
   - Dialog (6): DialogWarning, DialogError, DialogInfo, DialogQuestion, DialogSuccess, Shield
   - Window (4): WindowClose, WindowMinimize, WindowMaximize, WindowRestore
   - Action (14): ActionSave, ActionDelete, ActionCopy, ActionPaste, ActionCut, ActionUndo, ActionRedo, ActionSearch, ActionSettings, ActionEdit, ActionAdd, ActionRemove, ActionRefresh, ActionPrint
   - Navigation (6): NavBack, NavForward, NavUp, NavDown, NavHome, NavMenu
   - Files (5): FileGeneric, FolderClosed, FolderOpen, TrashEmpty, TrashFull
   - Status (3): StatusLoading, StatusCheck, StatusError
   - System (4): UserAccount, Notification, Help, Lock

   Derives: `Debug, Clone, Copy, PartialEq, Eq, Hash`. No serde -- IconRole is not serialized to TOML.
   Add `#[non_exhaustive]`.
   Add `pub const ALL: [IconRole; 42]` listing every variant.

2. **IconData enum** -- Two variants per research recommendation:
   - `Svg(Vec<u8>)` for SVG content
   - `Rgba { width: u32, height: u32, data: Vec<u8> }` for rasterized pixels

   Derives: `Debug, Clone, PartialEq, Eq`. No serde -- IconData is binary data, not serialized.
   Add `#[non_exhaustive]`.
   No tint metadata (connector's responsibility). No size hint on SVG (size is a load parameter).
   Use owned `Vec<u8>` (not Cow) to avoid lifetime parameter infection.

3. **IconSet enum** -- 5 variants: SfSymbols, SegoeIcons, Freedesktop, Material, Lucide.
   Derives: `Debug, Clone, Copy, PartialEq, Eq, Hash, Serialize, Deserialize`.
   Add `#[non_exhaustive]`.
   Add `from_name(&str) -> Option<Self>` method mapping kebab-case TOML strings.
   Add `name(&self) -> &'static str` method returning the kebab-case string.

4. **Wire up module** -- In `model/mod.rs`, add `pub mod icons;` and `pub use icons::{IconRole, IconData, IconSet};`.
   In `lib.rs`, add `IconRole, IconData, IconSet` to the `pub use model::{...}` line.
